These are simple, robust, thread-safe UDP and TCP servers suitable for many applications.

Package Specification

The {@link TcpServer} and {@link UdpServer} classes use the standard java.io classes with Streams and Datagrams and so forth. The {@link NioServer} class uses the java.nio classes and can handle TCP and UDP connections (and multiple ports) in a single server; it uses ByteBuffers.

Using These Servers in Your Projects

There's no need to (and you probably shouldn't) copy all of these source files to your projects or try to include them all in an attached jar file. You should copy whichever servers ({@link TcpServer}, {@link UdpServer}, and {@link NioServer}) to your project tree and give the file an appropriate package com.you.yourproject line at the top.

Licensing (Public Domain)

Since this is Public Domain, you don't need to worry about licensing, and you can simply copy the UdpServer.java or TcpServer.java files to your own package and use them as you like.
